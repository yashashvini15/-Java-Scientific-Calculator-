# Java Scientific Calculator

This is a scientific calculator built with Java Swing and operations.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Exponents and logarithms
- Trigonometric functions (sine, cosine, tangent)
- Statistical calculations (mean, median, mode)

## Building and Running

To build and run the project, you will need to have Java and Apache Maven installed on your system. Follow these steps:

1. Clone the repository to your local machine.
2. Open a terminal or command prompt and navigate to the project directory.
3. Run the command `mvn clean install` to build the project.
4. Run the command `java -jar target/scientific-calculator.jar` to start the calculator.
